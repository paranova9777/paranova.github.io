The purpose of this project was just for me to practice positioning player cameras, producing aiming machanics, and basic modeling of players and landscape. I producted a tank model within Blender and made mountain formation with one of the Unreal Editor tools. The bases of the game was is to just shoot other ememy AI within a certain time limit similar to World of Tanks.



I created this small level within Unreal Engine just to grab a basic concept of game engine API. The main work of the project is mainly backend with the coding. The coding delt with basic volume triggers to activate the certain events within the level such as sun positioning and audio. Plus, collitions meshes, textures and lighting was also a major part of the project.
